📖 Detailed Description (README Content)

This project is a Numerology Calculator Website built using HTML, CSS, and JavaScript. It helps users determine whether a name or mobile number is favorable based on numerology rules.

🔮 Features
Convert letters into numerological values
Calculate total and reduce to a single digit (1–9)
Display step-by-step breakdown of calculation
Highlight the final numerology number
Supports both Name and Mobile Number input
Includes Alphabet-to-Number mapping table
Displays Friend / Enemy / Neutral compatibility chart
